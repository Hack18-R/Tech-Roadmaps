Core Programming Concepts (Universal)

   - Data types (int, float, string, boolean, etc.)

3. Functions / Methods

   - Parameters and return values

4. Data Structures (Basics)
   - Arrays / Lists
   - Strings (manipulation, searching)

5. Object-Oriented Programming (OOP)
   - Classes and Objects
   - Encapsulation
   - Inheritance
   - Polymorphism

6. Error Handling
   - Exceptions / Try-Catch
   - Debugging basics

7. Input / Output (I/O)
   - Reading & writing data
   - Console I/O, File handling basics

8. Memory & Execution Basics
   - Stack vs Heap (conceptual level)
   - How programs run (compilation vs interpretation)

9. Problem Solving Mindset
   - Breaking problems into steps (algorithms)
   - Writing clean, efficient code
